# Discord Icon Assets

This directory contains assets related to the Antigravity Community Discord icon.

## Contents

- `icon.png`: The current Discord server icon.
- `prompt_screenshot.png`: A screenshot of the prompt used to generate the icon.

## Maintenance

To maintain future versions, please store the following files here:

- **Layered Source File**: If available (e.g., `.tif`, `.psd`), please save it as `icon_source.tif` (or appropriate extension).
- **Prompt Text**: Save the full prompt text in a file named `prompt.txt` or `prompt.md`.

## Generation Info

The icon was initially created using Nano Banana Pro.
